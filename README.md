# 项目简介

## 核心功能
- 浏览首页信息
- 作品展示
- 广告资讯
- 个人中心
- 用户管理
- 作品信息
- 类别管理
- 系统管理

## 适用场景
- 摄影爱好者交流
- 摄影作品展示与分享
- 摄影资讯获取
- 摄影社区管理

## 技术特点
- 使用Spring Boot框架，提高开发效率和系统稳定性
- 基于Vue.js的前端技术，提供良好的用户体验
- MySQL数据库存储，保证数据安全与可靠性
- Maven项目管理，简化构建过程
- Node.js用于构建前端应用，提高开发效率
- Java 8作为后端开发语言，支持多种开发需求
